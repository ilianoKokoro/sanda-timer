# Sanda Timer

<img src="https://raw.githubusercontent.com/ilianoKokoro/sanda-timer/refs/heads/main/core/src/main/res/mipmap-xxxhdpi/ic_launcher.webp" height="72">

**Sanda Timer** is a simple timer app for Wear OS and Android. It focuses on being lightweight, fast, reliable, and easy to use without unnecessary features getting in the way.

All timers are stored only on the device. Sanda Timer does not collect or sell personal data.

## Features ✨

- Create timers quickly on the go.
- Run multiple timers at the same time.
- Pause, resume, and cancel timers.
- Silent vibration for a discrete reminder in public.
- Designed with mordern Material 3 for Wear OS.
- Completely offline.
- No ads.
- No subscriptions.
