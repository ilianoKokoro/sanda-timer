Privacy Policy

Effective date: July 16, 2026

Sanda Timer ("the app") is developed by ilianoKokoro. This Privacy Policy explains how the app handles user information.

1. Information We Collect

Sanda Timer does not collect, transmit, sell, or share personal information.

The app stores timer information locally on the user's Wear OS device, including:

- Timer duration
- Timer state
- Timer start and end times
- Timer configuration required for operation

This information is stored only on the device and is not sent to the developer or any third party.

2. Data Backup

Sanda Timer supports Android system backup functionality. Depending on the device settings and the user's Android backup preferences, locally stored app data may be included in Android's automatic backup system and restored when setting up a device.

The developer does not directly access or control Android backup data.

3. How We Use Information

The app processes data only to provide timer functionality, including:

- Creating and managing timers.
- Restoring timer states.
- Scheduling accurate timer completion events.
- Showing notifications when timers finish.
- Providing vibration alerts.
- Running timer-related operations when required by the Android system.

4. Permissions

Sanda Timer uses Android permissions only for core app functionality:

- Notifications (`POST_NOTIFICATIONS`): Used to notify users when timers complete.
- Exact alarms (`USE_EXACT_ALARM`): Used to schedule precise timer completion events.
- Full-screen notifications (`USE_FULL_SCREEN_INTENT`): Used when a timer completion requires immediate user attention.
- Vibration (`VIBRATE`): Used to provide vibration feedback when timers finish.
- Boot completion (`RECEIVE_BOOT_COMPLETED`): Used to restore required timer functionality after device restart.
- Wake lock (`WAKE_LOCK`): Used to allow timer operations to complete reliably when the device is in a low-power state.

These permissions are not used to collect personal information.

5. Third-Party Services

Sanda Timer does not currently use:

- Analytics services
- Advertising services
- Tracking technologies
- User accounts
- Cloud storage services
- External data collection services

Future versions may include optional diagnostic services such as crash reporting. If data collection is introduced, this Privacy Policy and the Google Play Data Safety section will be updated accordingly.

6. Data Sharing

Sanda Timer does not share user data with third parties.

7. Children's Privacy

Sanda Timer does not knowingly collect personal information from children or any other users.

8. Data Deletion

All app data is stored locally on the user's device. Users can delete their data by:

- Clearing the app storage through Android settings.
- Uninstalling the application.

9. Security

Because Sanda Timer does not collect or store user data on external servers, there is no developer-controlled database containing user information.

10. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Changes will be reflected by updating the effective date.
